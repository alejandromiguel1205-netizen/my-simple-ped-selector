How to Add & Configure Peds

  • 1

    OpenIV Setup

    Place your custom files inside your dlc pack's peds.rpf folder:

    • Non-Streamed: Drop the 4 model files (.ydd, .ytd, .yft, .ymt) directly.
    • Streamed: Create a folder named exactly like the ped, and put the files inside.
  • 2

    Application Registry

    Open the app, press Ctrl + N, type the exact model name, select the gender, and set Is Streamed to True or False depending on your OpenIV structure.

  • 3

    Compile & Save

    Click the REBUILD! button to load the data into the database, and press Ctrl + G to successfully save your database.

Troubleshooting (Common Errors)

The game crashes instantly when spawning a ped

This happens when you exceed the game's internal limits. Make sure you have a custom gameconfig.xml, HeapAdjuster, and Packfile Limit Adjuster installed in your GTA V root directory.

The character is completely invisible or has broken textures

Double-check your Is Streamed setting in the selector. If a character has a texture subfolder in OpenIV, it MUST be set to True. If it's a single file, set it to False.
