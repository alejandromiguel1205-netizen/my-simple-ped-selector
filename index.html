Welcome!

Welcome to the official My Simple Ped Selector Help Center. This application was created to help you add custom addon characters (Peds) to your GTA V game in the easiest way possible, without having to manually edit complicated XML text files.

Follow this simple guide to learn how to place your character files in OpenIV and how to configure them inside the application to avoid errors.

1. How to put your Addons in OpenIV

Before using the application, you need to open OpenIV, turn on Edit Mode, and navigate to your custom addon DLC pack's folder, specifically inside peds.rpf. Check your character files and follow one of these two options:

Option A: Non-Streamed (Standard Ped)

Use this option if your character folder contains only 4 single files with these exact extensions:

• name.ydd
• name.ytd
• name.yft
• name.ymt

What to do: Drag and drop those 4 files directly into the root of peds.rpf.

Option B: Streamed (Complex Ped)

Use this option if your character is high-definition or has separate texture files/subfolders.

What to do: Right-click inside peds.rpf, create a new folder and name it exactly like your character. Then, open that folder and place all the character files inside it.

2. How to configure it in the App

Now that your files are inside OpenIV, open My Simple Ped Selector and follow these 3 steps to register your character:

1

Create the Profile

Go to the top menu and click Ped -> Create New Ped (or just press Ctrl + N on your keyboard). A window will open where you must fill in the information:

  • Ped Name: Type the exact name of your character's files in OpenIV. It must be identical!
  • Ped Gender: Choose if your character is Male (Hombre) or Female (Mujer).
  • Is Streamed:
    • Set to False if you followed Option A (dropped 4 loose files).
    • Set to True if you followed Option B (created a custom folder).

Click Save Ped to add it to the list.

2

Rebuild the Database

Look at the top panel of the application and click the big grey button that says REBUILD!. A green bar will slide loading your character into the game's internal database. Wait for the success message to pop up.

3

Save your Progress

Finally, go to File -> Save in the top menu or simply press Ctrl + G on your keyboard. This will permanently lock in your character. If you see an asterisk (*) in the window title, it means you have unsaved changes!

Application Shortcuts

You can use these handy keyboard shortcuts inside the app to manage your list much faster like a pro:

Action Shortcut Description
Create Ped Ctrl + N Opens the window to add a new character quickly.
Save All Ctrl + G Saves your entire character list workspace.
Save As File Ctrl + A Exports the selected character as a single .spm file to share it.
Open File Ctrl + O Imports a .spm character file directly into your list.
Duplicate Ctrl + D Clones the selected character adding -copy or numbers to the name.
Remove Delete / Supr Deletes the selected character after asking for confirmation.

How to Fix Common Errors

If you encounter any issues while playing or trying to spawn your character, check these common solutions:

The game crashes instantly when spawning the character!

This happens because GTA V has strict memory allocation limits for mods. To fix this, you must install an updated custom gameconfig.xml file, along with the HeapAdjuster and Packfile Limit Adjuster plugins inside your main GTA V folder. This expands the game memory limit.

The character is completely invisible or has glitched textures!

This is a mismatch error. Double-check your character's Is Streamed setting in the app list. If your character requires a folder structure inside OpenIV but you set it to False in the app, the game won't load the textures, making the model clip out or become invisible.
